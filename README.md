# quiz

Vanila JS를 사용하여 제작한 포트폴리오 페이지입니다. <br>
작성한 답을 LocalStorage에 저장한 후 JSON 파일에 있는 정답과 비교하여 마지막 결과 페이지에 출력할 수 있도록 코딩했습니다. <br>
또한 PWA로 구현하여 앱을 설치할 수 있습니다.

<br>
